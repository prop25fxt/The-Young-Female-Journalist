# Young Female Journalists Network (YFJN) — Static Website

This is a complete HTML/CSS/JS template designed for GitHub Pages.

## Deploy to GitHub Pages
1. Create a new GitHub repository (e.g., `yfjn-site`).
2. Upload all files/folders in this package to the repo root.
3. In GitHub: Settings → Pages → Deploy from branch → Select `main` (root).
4. Your site will be available at: `https://YOUR_USERNAME.github.io/yfjn-site/`

## Update placeholders
- Social handles: footer links in each HTML file.
- Email/phone: footer + contact page.
- Donation details: `donate.html`.
- Forms: connect to Formspree/Google Forms, or a backend (currently placeholders).
